1) Посмотреть где я                 -- pwd
2) создать папку                    -- mkdir foldername
3) Зайти в папку                    -- cd foldername
4) Создать три папки                -- mkdir foldername1 foldername2 foldername3 /  mkdir foldername{1..3} 
5) Зайти в любую папку              -- cd  foldername1
6) Создать 5 файлов (3 txt, 2 json) -- touch 1.txt 2.txt 3.txt name1.json name2.json / touch {1..3}.txt name{1..2}.json
7) Создать 3 папки                  -- mkdir 1 2 3
   Удалить любую папку / файл       -- rm 3.txt / rmdir 3
   Переименовать любую папку / файл -- git mv 3 4 / git mv 3.txt 4.txt 
8) Вывести содержимое файла         -- ls (если нужно вывести и скрытые файлы и файлы у которых перед названием стоит точка, то добавляем -а: -l - выводить подробный список, в котором будет отображаться владелец, группа, дата создания, размер и другие параметры; можно вывести общий вариант прописав ls -la)
9) Открыть любой txt файл           -- start 1.txt (nano 1.txt; vim 1.txt команды для открытия и редактирования файла в терминале)
10) Написать туда что-нибудь, любой текст -- После открытия файла (например командой vim 1.txt) нажимае 'i', печатаем текст 
11) Сохранить и выйти               -- нажимаем esc, затем :wq - команда для выхода с сохранением для vim, выйти без сохранения :q! / для nano - ctrl+"X" "Y" "enter"
12) Выйти из папки на уровень выше  -- cd ..
13) Переместить любые 2 файла, которые вы создали, в любую другую папку -- mv name1.json 1/name1.json
    mv 2.txt 1/2.txt (интересный момент, что при перемещении и копировании мы можем изменить название файла и его расширение. например можно написать mv 2.txt 1/2.docx таким образом мы переместили бы файл, поменяв его расширение)
14) Скопировать любые 2 файла, которые вы создали, в любую другую папку -- cp 1.txt 2/1.txt (+ так же , как и в перемещение, можно поменять расширение файла)
    cp name2.json 2/name12.json
15) Найти файл по имени             -- find -name 1.txt  (-name - по имени файла)
16) просмотреть содержимое в реальном времени -- cat 1.txt
17) вывести несколько первых строк из текстового файла -- head -2 1.txt
18) вывести несколько последних строк из текстового файла -- tail -2 1.txt
19) просмотреть содержимое длинного файла -- less 1.txt
20) Вывести дату и время            -- date
